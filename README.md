# 🪖 Soldier Monitoring System

A real-time military-style monitoring dashboard built using **MERN-inspired architecture (React + Node + MongoDB)**.  
The system tracks soldiers' health, risk levels, and live location with interactive dashboards, alerts, and analytics.

---

 Features

 Dashboard
- Live soldier data from MongoDB
- Risk-based analytics (LOW / MEDIUM / HIGH / PANIC)
- Interactive charts using Recharts
- Real-time updates every few seconds

 Live Map Tracking
- Soldier location tracking using React Leaflet
- Dynamic marker updates from backend
- Visual battlefield-style interface

 Alerts System
- Filters HIGH / MEDIUM / PANIC risk soldiers
- Real-time alert updates
- Risk radar visualization

 Command Center
- AI-based decision suggestions
- Risk classification engine
- Soldier monitoring cards

 Backend
- Node.js + Express API
- MongoDB database for storing soldier data 
- Seed script to generate sample soldiers

---

 Tech Stack

 Frontend:
- React.js (Vite)
- Recharts
- React Leaflet
- Axios
- CSS (custom military theme)

 Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
